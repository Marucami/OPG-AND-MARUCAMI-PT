## Как развернуть проект

 1. Убедитесь что установлен NodeJS версии 23+
```bash
node --version # v23.11.0
```
2. Разверните проект командой
```bash
npm ci
```
3. Запустить приложение командой
```bash
npm run start
```
4. Для того чтобы скомпилировать приложение
```bash
npm run make
```
